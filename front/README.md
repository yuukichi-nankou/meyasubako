

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```


## 開発者向けの動かし方
Webサーバの起動コマンド
```
$ go run web.go
```
http://127.0.0.1:8000
でアクセスできます。
